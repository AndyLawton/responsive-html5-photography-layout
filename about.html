What I know looks good

Update - Aug 9

  • Framework Changed. Images inside DL tags are invalid mark-up. Article Elements Added.
  • Internet Explorer 6, 7 and 8 now love it.
    • Older versions of IE will display at fixed 960 width.
    • Single column Layouts such as this will respond in all versions of ie. If you want IE6,7 and 8 to respond on resize you need respond.js - by Scott Jehl
  • I did not know the name attribute is obsolete. It still works but I'm looking into semantic alternative.
  • Changed some overall widths to make the type more robust.

Launch Date - Aug 5

  • Newest versions of firefox, chrome, safari on mac.
  • iPhone Landscape and Portrait Media Queries on Safari Browser.
  • HTML5 Shiv is included with html5boilerplate template. This allows old browsers to render html5 elements by creating and removing html5 elements with javascript before the browser loads the elements in html.
  • Html5 elements are styled display: block which applies default empty div styles to all Html5 Elements.
  • I always use a fallback of Unique Ids, just in case.
  • Smart Reusasble Measurements - All fonts are styled with ems. List Wrappers with % for Fluid Responsive Design
  • I wanted the browser width and height while I resized it. I wanted to easily see pixel break points but couldn't find an easy script that suited what I wanted so I made my own for you. See the Tool in action
  • Download Home Brewed Display Window Tool
